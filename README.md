# Tomato-Cultivars-Recognition-Using-pytorch

The images are labeled by the file prefixes. Labels correspond to the follow tomato cultivars:

01=>Kumato

02=>Beefsteak

03=>Tigerella

04=>Roma

05=>Japanese Black Trifele

06=>Yellow Pear

07=>Sun Gold

08=>Green Zebra

09=>Cherokee Purple

10=>Oxheart

11=>Blue Berries

12=>San Marzano

13=> Banana Legs

14=>German Orange Strawberry

15=>Supersweet 100
